# Release Notes (2.0.x)

## 2.0.0-M1

This is milestone release and is aimed at testing this new major version by early adopters. This is experimental. This release should not be used in production.

Java 17 is the minimum supported version. Scala 2.12 support has been dropped.

This milestone release is intended to be used with other Pekko 2.0 milestone module releases.

### Dependency Changes

* kafka-clients 4.1.2
